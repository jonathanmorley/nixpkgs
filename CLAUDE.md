# nixpkgs

After creating a PR in this repository, enable automerge on it (e.g., `gh pr merge --auto --squash`).

## Superpowers Plugin

When executing plans, always use the 'Subagent-Driven' execution option. Do not prompt for which execution method to use.
After creating a new superpowers skill or feature, follow up with a PR to the superpowers plugin repository to contribute the feature back upstream.
